# 🎓 EduTrack - Student Data Management Application

![Flutter](https://img.shields.io/badge/Flutter-02569B?style=for-the-badge&logo=flutter&logoColor=white)
![Dart](https://img.shields.io/badge/Dart-0175C2?style=for-the-badge&logo=dart&logoColor=white)

Aplikasi manajemen data siswa berbasis **Flutter**, memudahkan pengelolaan informasi siswa secara digital dan terstruktur.

---

## ✨ Fitur

- 🔐 Login & autentikasi pengguna
- 👨‍🎓 Tambah data siswa baru (nama, NIS/NISN, kelas, jenis kelamin, tempat lahir, alamat, no. telepon orang tua)
- 📋 Daftar siswa (Student List) dengan informasi ringkas
- 🔍 Manajemen data siswa per kelas
- 📱 Tampilan modern & responsif

---

## 🛠️ Teknologi

- **Framework:** Flutter
- **Bahasa:** Dart
- **Platform:** Android, iOS, Windows (multi-platform)
